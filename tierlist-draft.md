# Balatro Tier List 标注表 — 前八轮 vs 无限轮

**使用方法：** 每张卡牌后面有两个空位，分别填入 **前八轮** 和 **无限轮** 的档位（S/A/B/C/D/E）。
"当前"列是现在统一的档位，供参考。如果某张卡两个模式档位相同，可以直接填一样的。

格式：`当前档位 → 前八轮: __ / 无限轮: __`
