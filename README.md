## ✨ 核心功能

### 📚 卡牌图鉴

收录 **260+ 张**特殊卡牌的完整图鉴：

| 类型 | 数量 | 说明 |
|------|------|------|
| 🃏 Joker | 150 | 含效果、稀有度、协同关系、搭配建议 |
| 🔮 塔罗牌 | 22 | 强化/转换/经济效果 |
| 🪐 行星牌 | 12 | 升级各牌型等级 |
| 👻 灵魂牌 | 18 | 稀有特殊效果 |

支持多维度筛选：类型 / 稀有度 / 效果类型 / 适合牌型

### 🏆 小丑 Tier List

**150 张小丑牌**完整评级（S → E 六档）：

| Tier | 数量 | 定位 |
|------|------|------|
| **S** | 9 | 版本之子，无脑拿 |
| **A** | 29 | 极强，核心构筑基石 |
| **B** | 26 | 强力，大部分构筑优秀 |
| **C** | 38 | 可用，有合适的构筑能发挥 |
| **D** | 41 | 偏弱/场景局限 |
| **E** | 7 | 实在没牌了才选 |

点击任意卡牌可查看详细效果、数值和协同说明。

### 🎯 智能牌组匹配

选择你已有的卡牌 → 基于 **卡牌协同关系** 自动推荐最佳搭配。

- 推荐 Joker 搭配（含匹配度评分）
- 推荐塔罗/行星/灵魂牌
- 匹配 21 种构筑流派方向

### 💡 21 种构筑流派

| 流派 | 流派 | 流派 |
|------|------|------|

每种流派包含：核心卡牌推荐 / 搭配思路 / 进阶技巧 / 兼容流派
