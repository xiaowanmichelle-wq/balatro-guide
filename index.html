📖 新手攻略

🎯 基础概念

筹码 × 倍数 = 得分

Balatro 的核心公式：Chips × Mult = Score

  • 筹码(Chips): 分数的基础值，由牌型基础筹码 + 打出牌的点数 + 各种 +Chips 效果累加
  • 倍数(Mult): 加法型乘数，由牌型基础倍数 + 各种 +Mult 效果累加（如 +4、+8）
  • 倍率(XMult): 乘法型乘数，在倍数累加完成后，依次 ×XMult 相乘（如 X2、X3）

📐 计算顺序示例：

① 筹码 = 牌型基础 + 打出牌点数 + 各种 +Chips → 得到总筹码

② 倍数 = 牌型基础 + 各种 +Mult → 得到基础倍数

③ 倍率加成 = 基础倍数 × X2 × X3 ... → 得到最终倍数

④ 最终得分 = 总筹码 × 最终倍数

⚠️ 注意：倍数(+Mult)是加法叠加，倍率(XMult)是乘法叠加。1个 X2 倍率 ≈ 把你的倍数翻一倍，远比 +2 倍数强！

🔀 卡牌顺序很重要！

Balatro 中小丑卡牌和手中的扑克牌都可以手动拖拽调整顺序，这会直接影响计分结果。

  • 小丑(Joker)顺序：Joker 的效果按从左到右的顺序依次触发。拖拽 Joker 可以改变触发顺序
  • 扑克牌顺序：手牌中的扑克牌也可以拖拽排列，打出的牌按从左到右依次计分

🎯 顺序影响举例：

• 蓝图复制右侧的 Joker → 把最强 Joker 放在蓝图右边

• 头脑风暴复制最左侧的 Joker → 把最强 Joker 放在最左边

• 照片对打出的第一张人头牌 X2 → 人头牌尽量放最左边先打出

• +Mult 的 Joker 应放在 XMult 的 Joker 左边 → 先加后乘，收益最大化

💡 小技巧：长按/拖拽即可调整顺序。养成随时调整 Joker 排列的习惯，尤其是获得蓝图、头脑风暴等位置敏感的卡牌时！
